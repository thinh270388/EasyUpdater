# 🚀 EasyUpdater

**EasyUpdater** là thư viện cập nhật ứng dụng WPF/WinForms qua `.zip` — dễ tích hợp, không cần quyền admin, có UI và hoạt động tự động.

---

## ✨ Tính năng nổi bật

- ⬇️ Tải và cập nhật ứng dụng qua `.zip`  
- 🔃 Tự động kiểm tra version & khởi chạy lại app  
- 🪟 Giao diện WPF hiện tiến trình cập nhật  
- ✅ Không cần quyền admin hoặc cài đặt thêm  
- 🔧 Dễ cấu hình qua file JSON online  

---

## 🚀 Cài đặt

dotnet add package EasyUpdater.Core
